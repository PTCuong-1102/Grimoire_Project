The Citadel stands high above the clouds, a beacon of ancient power. Its walls, constructed from pure crystalline sapphire, refract the sunlight into a thousand dancing rainbows that span the valley below.

Before the [[Great Fracture]], this was the seat of the High Council. Now, only the wind whispers through the hollow halls. [[Arin]] believes that the core is still active, humming with a low frequency that only the magically attuned can hear.
